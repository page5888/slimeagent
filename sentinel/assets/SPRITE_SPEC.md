# 利姆路 Sprite 素材規格

## 基本規格
- **尺寸**：256×256 px
- **格式**：PNG，透明背景
- **風格**：像素風（pixel art）或 Q版（chibi）
- **朝向**：正面朝前

## 渲染順序（由下到上）

```
1. background（背景）
2. mount（載具）
3. body（本體，根據進化階段）
4. skin（皮膚覆蓋色）
5. core（晶核，胸口位置）
6. left_hand（左手持物）
7. right_hand（右手持物）
8. mouth（嘴巴表情）
9. eyes（眼睛）
10. helmet（頭盔）
11. vfx（環繞特效）
12. drone（跟隨精靈，偏右上）
13. frame（稀有度邊框，最上層）
14. title（稱號文字，圖片下方）
```

## 社群貢獻指南

我們歡迎任何人貢獻 sprite 素材！目前所有素材都是程式自動生成的 placeholder（色塊+文字），等待替換為真正的像素藝術。

### 如何貢獻

1. **Fork 此專案**
2. **選一個你想畫的 sprite**（參考上面的表格）
3. **按照規格作畫**：
   - 256×256 px PNG，透明背景
   - 像素風（pixel art）或 Q版（chibi）風格
   - 正面朝前
   - 裝備圖層需對齊本體位置（疊上去要合理）
4. **放到正確的資料夾**：`sentinel/assets/sprites/{slot}/{filename}.png`
5. **提交 Pull Request**，標題格式：`art: add {sprite_name}`

### 注意事項

- 一個 PR 可以包含多個 sprite
- 請勿修改 placeholder 產生器或渲染程式碼（除非是 bug fix）
- 如果同一個 sprite 有多人同時在畫，先合併的優先
- 素材一旦合併即以 MIT License 授權

### Placeholder 圖辨識

Placeholder 圖片特徵：
- 檔案大小通常 < 5KB
- 圖片上有 "PLACEHOLDER" 浮水印
- 用色塊+形狀代替真正的圖案

真正的 sprite 會自動取代 placeholder（渲染器優先使用存在的檔案）。

### 風格參考

- 基礎風格：日系像素風，類似《轉生史萊姆》Q 版
- 色調：偏亮，但高稀有度可以暗色系
- 表情：可愛為主，魔王級以上可以帥氣/霸氣
- 特效：用半透明圖層表現發光、粒子等效果
