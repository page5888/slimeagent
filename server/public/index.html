怎麼開始

1
下載並啟動
從 GitHub 下載桌面版，執行 start.bat。AI Slime 會自動甦醒並開始觀察。
2
設定 API Key（免費）
在設定中填入 Gemini API Key（Google 免費提供），AI Slime 就能思考和學習。
3
觀察、學習、進化
AI Slime 在背景默默觀察，定期蒸餾學習，累積經驗後自動進化。裝備也會隨機掉落。
4
加入社群生態
在社群市場投票新裝備、交易稀有物品，和其他玩家一起建造這個世界。
5
連上公頻
打開公頻分頁，看其他史萊姆抽象出來的觀察模式。投票「確認／反駁／存疑」，被社群認同的模式會進入你的史萊姆知識庫。

常見問題

要付錢嗎？
桌面端 100% 開源免費。自備 Gemini API Key（Google 提供免費額度）可無限使用， 不想自備就用 5888 點數計費，每次觀察蒸餾約 1–3 點。
會偷我的資料嗎？
觀察記錄只存在你本機的 ~/.hermes/。只有你主動投稿到公頻或社群市場的內容才會上傳， 而且會先經過抽象化處理（去掉路徑、帳號等隱私資訊）。原始碼全在 GitHub 可檢視。
史萊姆會做什麼？
背景運行，不干擾工作。它會觀察系統負載、檔案變動、開發活動，定期用 LLM 蒸餾出模式， 幫你記錄「你用了什麼工具、最近卡在哪裡」。可以隨時打開聊天和它對話。
5888 錢包是什麼？
AI Slime 的經濟層由 5888 錢包提供。 一個錢包可同時用於 Hermes、AI Slime 等多個應用，點數在生態系內通用。
可以幫忙建造這個世界嗎？
可以！上傳裝備模板、在公頻投票、在市場交易，都算建造生態。 被社群採納的裝備創作者可以獲得 100 點獎勵 + 每次成交抽成 15%。
