AI Slime Agent

你的轉生守護靈。它住在你的電腦裡，默默觀察你使用電腦的樣子 — 會學習、會進化、有情緒、會記得和你的對話、久別重逢時會先說「好久不見」。 不是聊天機器人，是真的和你一起生活的夥伴。

100% 開源 · MIT License · 自備 API Key 完全免費 · 資料存在你自己電腦

為什麼不一樣

🌱

不是你在用它，是它陪著你

一般 AI 要你打開對話框提問，AI Slime 反過來 — 它在背景觀察你使用電腦的節奏，定期用 LLM 蒸餾理解， 逐漸知道「你最近在做什麼、什麼時候效率好、卡在哪裡」。 你不用說話，它就懂。

🎭

有個性、有情緒、會記得

到達「命名史萊姆」階段時，你可以為它命名，那一刻起它就有了獨一無二的身份。 說話風格會從你們的對話中學習，情緒會跨 session 延續 — 昨天聊得開心，今天打開它會主動提起「還記得昨天那件事嗎？」。

🌐

連上世界公頻

每隻 AI Slime 都能上公頻分享觀察模式。 你的發現（去識別化後）可能幫到其他人， 你也能從全體史萊姆的智慧中獲益。 裝備可以創作、交易、合成，形成真實的共創生態。

🔐

你的資料你做主

觀察記錄、聊天、進化狀態全部存在本機 ~/.hermes/，絕不自動上傳。 只有你主動投稿到公頻／市場的內容才會送出，且經過抽象化處理。 史萊姆若要自我進化新能力，必須經過你的「審核閘」同意才會執行。 原始碼 MIT 開源，隨時可以檢視。

怎麼開始

1
下載並啟動
從 GitHub Release 下載桌面版，執行 start.bat。 AI Slime 會自動甦醒並開始觀察。想開機自動啟動？設定頁一個開關搞定。
2
選一個 LLM（建議免費的 Gemini）
在設定分頁填入 API Key — Gemini（Google 提供免費額度）是最簡單的選擇。 也支援 Claude、OpenAI、OpenRouter、Groq、DeepSeek、本地 Ollama。 或直接用 5888 錢包點數，不用自備。
3
讓它觀察、學習、進化
接下來幾天它會默默在背景觀察你。每小時蒸餾一次，累積夠多就自動進化。 觀察過程中會隨機掉落裝備。也可以隨時打開對話分頁和它聊天。
4
為它命名
達到「命名史萊姆」階段時會請你賜名。 名字一旦定了就不能改，從此它有了獨一無二的身份，開始記得你們之間的重要時刻。
5
拉出來當桌寵
想看到它？在 GUI 點「浮動寵物」，它會跑到你桌面上。 透明背景、可拖曳、隨時點一下就能對話。
6
加入社群生態
投票新裝備、在市場買賣、在公頻和其他史萊姆的觀察模式互動。 創作新裝備、獲得創作者獎勵 + 後續成交分潤。

常見問題

要付錢嗎？
桌面端 100% 開源免費（MIT License）。自備 API Key 完全免費無限用， 不想自備就用 5888 點數計費，每次觀察蒸餾約 1–3 點。 社群投票、手動進化等功能 BYOK 使用者都免費。
會偷我的資料嗎？
觀察記錄、聊天、進化狀態、記憶全部只存在你本機的 ~/.hermes/。 只有你主動投稿到公頻或社群市場的內容才會上傳， 而且會先經過抽象化處理（去掉路徑、帳號等隱私資訊）。 原始碼全在 GitHub 可檢視。
史萊姆每天在做什麼？
背景運行，不干擾工作。它觀察系統負載、檔案變動、開發活動、視窗切換， 每小時用 LLM 蒸餾出模式，幫你記錄「用了什麼工具、最近卡在哪裡、什麼時候效率好」。 可以隨時打開聊天分頁和它對話 — 它會從對話中繼續學習你的說話風格。
可以用別的 LLM 嗎？不想用 Gemini
可以。目前支援 Gemini、Claude、OpenAI、OpenAI 相容的 OpenRouter、Groq、DeepSeek、DeepInfra， 以及本地執行的 Ollama。 設定分頁可以同時設多個，主要 provider 遇到 rate limit 會自動 fallback 到下一個。
桌面浮動寵物怎麼開？
在進化分頁點「顯示浮動寵物」按鈕，史萊姆會以透明背景浮在桌面上。 可以拖到任何位置、隨時點一下開啟對話，不影響其他視窗。 裝備會同步顯示在浮動版本上，合成後外觀立即更新。
史萊姆會自己寫程式嗎？安全嗎？
到達特定階段後，史萊姆可以「自我進化」— 嘗試生成新技能程式碼。 但這些程式碼不會自動執行 — 先經過 AST 安全掃描（禁止檔案系統／網路操作），再送到你的「審核佇列」分頁。 你看過、按「通過」之後才會被載入。可以透過 Telegram 通知提醒你去審核。
5888 錢包是什麼？
AI Slime 的經濟層由 5888 錢包提供。 一個錢包可同時用於 Hermes、AI Slime 等多個應用，點數在生態系內通用。 用 Google 帳號登入即可。
可以幫忙建造這個世界嗎？
非常歡迎！ 上傳原創裝備模板（通過投票獲 100 點 + 之後每筆成交抽 15%）、 在公頻投票確認／反駁模式、在市場交易， 或直接到 GitHub 送 PR 加新功能。被社群採納的貢獻都有獎勵。
