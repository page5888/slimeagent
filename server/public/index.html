功能特色

怎麼開始

1
下載並啟動
從 GitHub 下載桌面版，執行 start.bat。AI Slime 會自動甦醒並開始觀察。
2
設定 API Key（免費）
在設定中填入 Gemini API Key（Google 免費提供），AI Slime 就能思考和學習。
3
觀察、學習、進化
AI Slime 在背景默默觀察，定期蒸餾學習，累積經驗後自動進化。裝備也會隨機掉落。
4
加入社群生態
在社群市場投票新裝備、交易稀有物品，和其他玩家一起建造這個世界。
